2001 game

Play the game 2001 with your computer

Rules of the game

1. Each player starts with a number of points equal to 0.
2. In his/her turn, a player rolls 2 dice (standard six-sided dice).
3. The number of meshes thrown is added to the total number of points.
4. Starting from the second turn:
    if the player throws a 7, he divides his number of points by this value discarding the fractional part,
    if he throws out 11, he multiplies the current number of points by this value.
5. The player who gets 2001 points first wins.
